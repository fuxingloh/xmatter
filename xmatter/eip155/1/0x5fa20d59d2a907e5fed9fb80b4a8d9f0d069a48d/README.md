Noggles (NOGS) is a nounish coin created to push the boundaries of the ⌐◨-◨ meme.
Launched on July 4th 2023 and powered by Nouns Name Service (NNS) and by other contributors, Noggles is an experimental attempt to create a nounish economy around the proliferation of the iconic ⌐◨-◨.
